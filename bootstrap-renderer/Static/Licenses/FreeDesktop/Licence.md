## service-login.ogg
**Descripción:**  
Sonido utilizado para indicar que un servicio o aplicación ha iniciado sesión correctamente.  
**Archivo (Linux):** `/usr/share/sounds/freedesktop/stereo/service-login.ogg`
**Archivo (Project):** `../../Sounds/service-login.oga`

## service-logout.ogg
**Descripción:**  
Sonido utilizado para indicar que un servicio o aplicación ha cerrado sesión o terminado un proceso.  
**Archivo (Linux):** `/usr/share/sounds/freedesktop/stereo/service-logout.ogg`
**Archivo (Project):** `../../Sounds/service-logout.oga`
