# StepLauncher – Code of Conduct

## 1️⃣ Propósito

StepLauncher busca ser un **launcher seguro, divertido y profesional para Minecraft**, donde todos los usuarios y colaboradores puedan interactuar, jugar y contribuir sin preocupaciones.
Este Código de Conducta también establece que StepLauncher puede **ser distribuido libremente**, siempre y cuando **se dé crédito al autor original y se respeten las licencias** de las herramientas y proyectos que se utilicen dentro del launcher.


## 2️⃣ Comportamiento esperado

Se espera que todos los usuarios, colaboradores y participantes:

* Respeten a todos los miembros de la comunidad y contribuyentes.
* Mantengan un lenguaje cordial y apropiado en todos los canales oficiales (Discord, GitHub, Reddit, X / Twitter,foros, etc.).
* Reporten bugs, problemas o sugerencias de manera **constructiva y clara**.
* Den crédito apropiado a terceros cuando utilicen su trabajo o código como referencia o inspiración.
* Protejan la privacidad y seguridad de todos los usuarios.


## 3️⃣ Comportamiento inaceptable

No se permitirá:

* Compartir exploits, cheats, código malicioso o contenido que pueda dañar la experiencia del launcher.
* Difundir contenido ilegal o inapropiado.
* Suplantar a otros usuarios o desarrolladores.
* Violación de licencias de software o EULA de terceros.


## 4️⃣ Consecuencias

El incumplimiento de este Código de Conducta puede resultar en:

* Advertencia privada o pública.
* Suspensión temporal de participación en Discord o contribuciones al repositorio.
* Bloqueo permanente en casos graves, como publicar exploits, malware o vulnerar la seguridad de otros usuarios.
