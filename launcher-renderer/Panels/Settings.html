Avanzados

Utilizar archivos compartidos

Esta opción crea una carpeta llamada Shared dentro de .StepLauncher. Allí se almacenarán los assets, librerías y Java. La primera instalación puede tardar un poco más, pero las siguientes serán muchísimo más rápidas, ya que reutilizan estos archivos en lugar de descargarlos nuevamente. Esto acelera tanto la creación de instancias como el inicio del juego (también se aplica a las instancias existentes, y solo se aplica a proximas instalaciones).

Aceleración de Hardware

Optimiza el renderizado usando la GPU. Puede mejorar el rendimiento en algunos sistemas.

Uso de GPU

Define qué tarjeta gráfica utilizará el juego. Automático elige la mejor opción según tu sistema. Usa GPU dedicada para máximo rendimiento o integrada para menor consumo de energía.

Optimizador de memoria (GC)

Configura cómo Java gestiona la memoria del juego. Automático selecciona la mejor opción. Los modos avanzados pueden mejorar el rendimiento o reducir lag en equipos potentes.
